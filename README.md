# 📝 To-Do List App

## 📌 Project Overview
An interactive **To-Do List application** to manage daily tasks.  
Users can add, delete, and mark tasks as completed.

## ✨ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Interactive and responsive UI
- Works on desktop and mobile

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
